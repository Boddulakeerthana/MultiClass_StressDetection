# MultiClass_StressDetection Through Heart Rate Variability
This project focuses on detecting stress or no stress using Heart Rate Variability (HRV) data. The system analyzes physiological signals and applies machine learning techniques to identify whether a person is experiencing stress or is in a normal state.

Unlike multiclass prediction, this project works on binary classification, making the model simple, efficient, and suitable for real-world health monitoring applications.
